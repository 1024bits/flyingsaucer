Welcome to Eeze

Eeze is controlled through keyboard commands exclusively, listed below.

To return to the demos, press ALT-N to advance to the next in sequence.

ActionKeystroke
Help screenALT-H
Increase font sizeCTRL-I
Reset font sizeCTRL-0 (zero)
Decrease font sizeCTRL-D
Expand screen sizeALT-L
Shrink screen sizeALT-S
Next demo in listALT-N
