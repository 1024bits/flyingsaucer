The Flying Saucer Team is proud to present Beta Release 4 (R4) of our 100% Java library for rendering XHTML + CSS2.1 including tables, forms, the box model, lists, selectors, images, and text formatting.

This release brings updates to the infrastructure with:

  • a new package heirarchy
  • a new unified CSS module
  • automated unit tests to validate our rendering compliance
  • new configuration and logging subsystem
  • lots more code cleanup and documentation
  • separate browser and about box sub projects

This release paves the way for the new features and speed increases coming in R5. Thanks for all of your support and please send us your feedback.

The Flying Saucer XHTMLRenderer can be used anywhere a program needs rich styled content such as ebook readers, chat apps, splash screens, help viewers, report generation, web-enabled thick clients, and online music stores.
