The Flying Saucer Team proudly presents version 0.003 of Flying Saucer, the Open Source 100% Java XHTML Renderer.

This release introduces:

  • Text selection with the mouse
  • XHTML Validation
  • A new About Box component (which you are watching right now :)

And the following bug fixes

  • Better form item alignment
  • browser loads demos via the web
  • removed signed jar in webstart app
  • The first speed tests for rendering
