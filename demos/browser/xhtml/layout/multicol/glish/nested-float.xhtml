LAYOUT TECHNIQUES: nested float

A Liquid Layout

This page is based on an example in CSS Layout Techniques, from glish.com, a resource for web developers and designers. We include it here to show how Flying Saucer handles liquid layouts--without XHTML tables!

Your menu items go on the right side. Try resizing the browser window to see how the page reflows nicely--this is a liquid layout. The menu on the left is specified as floating, so the page should re-flow nicely when you resize the browser.

This is a very simple layout that features a nested, floated menu in the upper right. Easily reversed. A variation of this technique is in use on the glish.com home page.
