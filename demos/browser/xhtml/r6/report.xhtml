Tables Support
Tables support includes: Tables support is pretty basic at this point. We support the table, th, tr and td tags. Table elements can have styles attached to them--in this demo, to show an alternating row highlight color. In-cell alignment works, as do borders. Colspan, rowspan, cellpadding and cellspacing attributes are supported. Table and column widths can be set to a specific or proportional (em, %) value, but the current layout algorithm is not very far advanced. More work to do--but as this page shows, you can display simple tables to good effect.
