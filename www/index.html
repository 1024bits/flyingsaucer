News

The Flying Saucer team is proud to present Beta Release 3 of XHTMLRenderer, a 100% Java library for embedding rich XHTML and CSS content in desktop applications. It currently supports most of CSS2.1 including tables, forms, the box model (borders, backgrounds, padding, margins), lists, advanced selectors, images, and text formatting (fonts, italics, vertical alignment, etc.).

This release introduces:

  • A new CSS module for 10 times the rendering speed.
  • A new browser application with demo pages.
  • An easy to use scrolling About Box you can add to your own programs with 3 lines of code.
  • A slew of bug fixes and more compliance with the CSS 2.1 standard.

The Flying Saucer XHTMLRenderer can be used anywhere a program needs rich styled content such as ebook readers, chat apps, splash screens, help viewers, report generation, web-enabled thick clients, and online music stores.

We are actively looking for new contributors and users. Please join the mailing list and visit us at http://xhtmlrenderer.dev.java.net/
