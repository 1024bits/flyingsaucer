Flying Saucer: The 100% Java XHTML + CSS 2 renderer

Flying Saucer is a 100% Java XHTML renderer with a focus on strict compliance with the complete W3C CSS 2.1 standard, mainly for thick clients like the iTunes Music Store, chat programs, help viewers, ebook readers, splashscreens, and any other place where html is useful inside an application. Flying Saucer is not a general purpose webbrowser. Legacy and non-compliant pages (ie, the general internet) will be supported through compatibility libraries rather than modifying the core renderer. We aim to be the definitive open 100% Java renderer for XHTML content.
